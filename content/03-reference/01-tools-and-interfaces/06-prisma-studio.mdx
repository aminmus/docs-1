<TopBlock>

Prisma Studio is a visual editor for your database. To run Prisma Studio, run the following command in your terminal:

```console
npx prisma studio --experimental
```

> **Note**: Prisma Studio replaces Prisma Admin, and is only compatible with Prisma 2. Be aware that Prisma Studio is currently in an **experimental state**.

</TopBlock>

## Troubleshooting

### Prisma Studio queries return error or no result

Caching issues may cause Prisma Studio to use an old version of the query engine even after an update. To resolve, delete the following folders:

* `~/.cache/prisma` on macOS and Linux
* `%AppData%/Prisma/Studio` on Windows
